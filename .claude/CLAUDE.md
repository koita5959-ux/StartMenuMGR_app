# StartMenuMGR - ClaudeCode向けプロジェクト情報

## プロジェクト概要

Windows 10 スタートメニュー管理ツール（RPAシリーズ / DesktopKit関連）。
疑似スタートメニュー方式で、アプリ内にスタートメニュー構造を再現し、「適用」で実フォルダに反映する。

## フォルダ構成

```
StartMenuMGR_app/
├── Assets/                        ← アイコン（人が触る）
├── _works/                        ← ドキュメント（人が触る）
├── build.bat                      ← ビルド一括実行（バージョン変数はここで管理）
├── ご利用にあたって.txt             ← 利用者向け（配布物）
├── StartMenuMGR_setup_vXX.XX.exe  ← インストーラー（配布物）
├── StartMenuMGR_vXX.XX.zip        ← 配布ZIP
└── StartMenuMGR/                  ← ソースコード（開発者の領域）
```

直下は人が触る領域。ソースコードは `StartMenuMGR/` 内に閉じ込める。

## ビルド〜インストーラー（一括実行）

プロジェクトルートで `build.bat` を実行する。
バージョン番号は `build.bat` 冒頭の `APP_VERSION` で管理する。

```bat
build.bat
```

処理内容: dotnet publish → Inno Setup コンパイル → 配布ZIP作成（3ステップ、エラー時は中断）

## 開発ルール

- **Gitコミットはユーザーが指示する。** 勝手にコミットしない
- **起動方法は2つだけ。** `dotnet run`（開発中）か インストーラー経由（テスト）。bin/Publish直接起動は使わない
- **トラブル時は作業を止めて状況報告。** 修正はユーザー承認後に実行
- **壊れた箇所以外のコードを触らない。** 「ついでに直す」はしない
- **インストーラーの品質 = アプリの信頼性。** DesktopKit方式がゴール像（参考資料あり）
- **基準書を特定アプリの都合で変えない。** アプリ固有の話は制作指示書に書く
