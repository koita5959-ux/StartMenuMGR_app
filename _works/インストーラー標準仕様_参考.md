## DesktopKitインストーラーの構成

起動時に環境チェックを行い、結果に応じて3つの選択肢を提示する方式。

### 環境チェック結果（表示例）
```
.NET Desktop Runtime v8.0.24 : インストール済み
DesktopKit v1.0.0 : インストール済み
```

### インストール選択肢（3択）

| 選択肢 | 内容 |
|--------|------|
| 推奨インストール | .NET Desktop Runtime と アプリ本体をインストール |
| 本体のみインストール | アプリ本体のみ（上書き更新）。ランタイムは別途必要 |
| アンインストール | アプリをアンインストール |

### この方式の意図（基準書との対応）

- 環境チェック → 「利用者のことを意識したリリース」（基準書 1-2）
- ランタイム検出＋案内 → 「動作環境の明示」（基準書 5-1）
- アンインストール選択肢 → 「痕跡を残さない」（基準書 4-2）

基準書の思想を忠実に実装すると、自然とこの形に至る。
